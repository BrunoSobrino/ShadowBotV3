### `INSTALACION TERMUX`
```bash
> termux-setup-storage
> pkg update && pkg upgrade -y -y
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> git clone https://github.com/BrunoSobrino/ShadowBotV3
> cd ShadowBotV3
> npm install
> npm install -g npm@8.5.4
> npm update
> npm start
```
